# Multi-Agent AI Research Assistant
An Agentic AI system that automatically researches topics, gathers information from the internet, and generates structured research reports using multiple AI agents.

## Features
- Planner Agent creates a research strategy
- Research Agent retrieves information from the web
- Writer Agent generates a structured report
- Automated report saving
- Modular AI agent architecture

## Architecture
User Input
    ↓
Planner Agent
    ↓
Research Agent
    ↓
Writer Agent
    ↓
Final Research Report

## Tech Stack
- Python
- LangChain
- Groq LLM
- Tavily Search API
- Environment Variables

## Example Output
The system generates structured research reports including:
- Introduction
- Key Concepts
- Important Insights
- Conclusion
